# Безопасно.Форум – лендинг приглашения

Этот репозиторий хранит публичный лендинг первого юридического форума «Безопасно.Форум» с React, Tailwind CSS и плагином TailGrids. Страница посвящена I открытому юбилею БЕЗОПАСНОСДЕЛКА™ и собирает регистрацию в официальном боте с упором на безопасность сделок и тёплый нетворкинг.[^1]

## Текущий стек
- React 19 + Vite 7 для быстрого SPA.[^2]
- Tailwind CSS 3.4 с `tailgrids/plugin` и кастомной анимированной градиентной темой для hero-блока.[^1][^2]
- Vitest + React Testing Library для простых UI-проверок без Jest.[^2]

## Локальный запуск
```bash
cd web
npm install
npm run dev
# тесты
npm test
```

## Guardrails и процессы
- Файл `content/about.md` редактируется только после отдельного согласования с заказчиком — использовать как read-only источник.[^3]
- Перед любыми изменениями кода нужно вести исследование и фиксировать находки в `research/RESEARCH_STEP.md`.[^1]
- Все решения и ход работ документируем в `INFO.md` и `TODO.md`; завершённые задачи переносятся в архив `TODO.md`.[^1]

[^1]: research/RESEARCH_STEP.md  
[^2]: web/package.json  
[^3]: INFO.md
